# 🌐 SocialSphere — Full-Stack MERN Social Media Platform

A production-ready, portfolio-worthy social media platform built with:
- **React.js** + Redux Toolkit + Framer Motion
- **Node.js** + Express.js + Socket.IO
- **MongoDB** + Mongoose
- **Cloudinary** for image uploads
- **JWT** authentication + **bcrypt** password hashing
- **Docker** + **Nginx** for deployment

### Prerequisites
- Node.js 18+
- MongoDB Atlas or local MongoDB
- Cloudinary account (free tier works)

### 1. Clone & Install
```bash

# Install backend
